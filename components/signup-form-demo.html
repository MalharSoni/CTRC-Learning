Signup Form Component

A beautiful, customizable signup form built with Framer Motion animations and the CTRC design system. Features smooth transitions, input validation, and social authentication options.

  • Smooth Framer Motion animations on form elements
  • CTRC design system integration (Inter font, accent colors)
  • Input focus states with accent ring effects
  • Social authentication buttons (GitHub, Google, Discord)
  • Responsive grid layout for name fields
  • Phosphor Icons for social buttons
  • Gradient accent on submit button with hover glow
  • Form validation support built-in

Usage Example

import { SignupForm } from './signup-form'

<SignupForm
  title="Join CTRC Learning"
  subtitle="Start your robotics journey today"
  onSubmit={(data) => {
    console.log('Form data:', data)
    // Handle signup logic
  }}
  showSocialAuth={true}
/>
