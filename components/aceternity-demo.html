Aceternity UI Components

Three premium UI components adapted from Aceternity UI for the CTRC Learning platform. Built with vanilla TypeScript, optimized for performance, and fully customizable.

01. Text Hover Effect

A bold text display with an animated stroke effect on hover. Perfect for hero sections and attention-grabbing headlines.

Live Demo

Hover over the text to see the glow and stroke animation.

Usage

<!-- HTML -->
<div id="hero-text" data-text-hover-effect data-text="CTRC" data-stroke-color="#00ff88"></div>

<!-- Or with JavaScript -->
<script type="module">
  import { TextHoverEffect } from './components/text-hover-effect.js';

  new TextHoverEffect('#hero-text', {
    text: 'CTRC',
    duration: 0.5,
    strokeColor: '#00ff88'
  });
</script>

Options

  • text - The text to display (required)
  • duration - Animation duration in seconds (default: 0.5)
  • strokeColor - Stroke/glow color (default: #00ff88)
  • fillColor - Fill color on hover (default: transparent)

03. Dotted Glow Background

An animated canvas-based background with pulsing, glowing dots. Creates depth and visual interest without being distracting.

Live Demo

Watch the dots pulse and glow at varying speeds.

Master Robotics

Build competition-ready robots with hands-on curriculum designed for VEX V5.

Usage

<!-- HTML -->
<div id="hero-section" data-dotted-glow-background
     data-gap="15"
     data-radius="1.8"
     data-color="rgba(255, 255, 255, 0.15)"
     data-glow-color="rgba(0, 255, 136, 0.6)"
     data-speed-min="0.3"
     data-speed-max="1.6">
  <h1>Your Content Here</h1>
</div>

<!-- Or with JavaScript -->
<script type="module">
  import { DottedGlowBackground } from './components/dotted-glow-background.js';

  new DottedGlowBackground('#hero-section', {
    gap: 15,
    radius: 1.8,
    color: 'rgba(255, 255, 255, 0.15)',
    glowColor: 'rgba(0, 255, 136, 0.6)',
    speedMin: 0.3,
    speedMax: 1.6
  });
</script>

Options

  • gap - Distance between dot centers (default: 12)
  • radius - Base radius of each dot (default: 2)
  • color - Dot color (default: rgba(0,0,0,0.7))
  • glowColor - Shadow/glow color (default: rgba(0, 170, 255, 0.85))
  • opacity - Global opacity multiplier (default: 0.6)
  • speedMin - Minimum pulse speed (default: 0.4)
  • speedMax - Maximum pulse speed (default: 1.3)
  • speedScale - Global speed multiplier (default: 1)

Installation

1. Include the CSS

<link rel="stylesheet" href="stylesheets/aceternity-components.css">

2. Include the Component Scripts

<script type="module" src="components/dotted-glow-background.js"></script>
<script type="module" src="components/card-spotlight.js"></script>
<script type="module" src="components/text-hover-effect.js"></script>

3. Use Data Attributes (Auto-initialization)

Components will automatically initialize on elements with the appropriate data attributes.

4. Or Use JavaScript API

import { DottedGlowBackground } from './components/dotted-glow-background.js';
import { CardSpotlight } from './components/card-spotlight.js';
import { TextHoverEffect } from './components/text-hover-effect.js';

// Initialize programmatically
const bg = new DottedGlowBackground('#hero', { gap: 15 });
const card = new CardSpotlight('.feature-card', { radius: 400 });
const text = new TextHoverEffect('#title', { text: 'CTRC' });
