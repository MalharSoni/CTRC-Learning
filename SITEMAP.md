## Page Types

### Homepage (`index.html`)
- **Purpose:** Landing page, first impression
- **Components Used:**
  - Canvas Background (mouse-reactive grid)
  - User Type Selector (Student/Educator/Mentor)
  - Course Cards (Feature Section style)
  - Footer with navigation
- **Key CTAs:**
  - Start Learning → `/foundation/`
  - View Projects → `/projects/`
  - Browse Handbook → `/design-handbook/`

### Unit Overview Pages (`unit*/index.md`)
- **Purpose:** Introduce unit, show progress
- **Components Used:**
  - Timeline (lesson progress)
  - Feature cards (lesson highlights)
- **Structure:**
  - Unit objectives
  - Lesson list
  - Challenge list
  - Prerequisites
  - Learning outcomes

### Lesson Pages (`*.md`)
- **Purpose:** Teach specific skill or concept
- **Structure:**
  - Learning objectives
  - Prerequisites
  - Step-by-step instructions
  - Diagrams/images
  - Code examples (for programming)
  - Practice exercises
  - Next lesson link

### Challenge Pages (`challenge-*.md`)
- **Purpose:** Hands-on competition/project
- **Structure:**
  - Challenge brief
  - Rules
  - Scoring criteria
  - Time limit
  - Materials needed
  - Judging rubric
  - Success criteria

### Project Overview (`projects/*/index.md`)
- **Purpose:** Introduce project, motivate students
- **Structure:**
  - Project description
  - What you'll build
  - What you'll learn
  - Prerequisites
  - Module list
  - Final deliverable

### Design Handbook Pages (`design-handbook/*/index.md`)
- **Purpose:** Quick reference, lookup
- **Structure:**
  - Topic overview
  - Key concepts
  - Formulas/calculations
  - Diagrams
  - Real examples
  - Common mistakes
  - Quick tips
