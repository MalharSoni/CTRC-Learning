## Image Usage Guidelines for CTRC Curriculum

### Permitted Uses
- Educational instruction within CTRC Learning programs
- Student reference materials
- Internal training documentation
- Non-commercial educational presentations

### Required Attribution Format

For Creative Commons images:
```
[Image description]
Source: [Author/Creator Name]
License: [License Type]
URL: [Source URL]
```

For product photos:
```
[Product description]
Photo courtesy of [Company Name]
URL: [Source URL]
Used with permission for educational purposes
```

### Image Processing Before Use
1. Resize to max 1200px width for web use
2. Optimize file size (use JPEG quality 85%)
3. Add watermark if required: "CTRC Educational Use Only"
4. Rename descriptively: `[archetype]-[source]-[number].jpg`

## Notes for Manual Download

**Why Manual Download is Needed:**
Many of these sites (Thingiverse, Printables, Instructables) have bot protection that prevents automated scraping. Images must be downloaded manually by visiting each URL.

**Recommended Workflow:**
1. Open each URL in browser
2. Right-click on relevant images
3. Save with descriptive names
4. Place in appropriate folder
5. Update ATTRIBUTION.md with details

**File Naming Convention:**
```
[archetype]-[source]-[number].jpg

Examples:
- drum-spinner-alexkorvin-1.jpg
- vertical-disc-drizzle-1.jpg
- horizontal-bar-beater-1.jpg
- undercutter-lordtoby-1.jpg
- midcutter-dinnertime-1.jpg
- n20-motor-datasheet.pdf
- n20-d-shaft-fit-1.jpg
```
