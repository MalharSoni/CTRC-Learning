# BattleBots Curriculum Images

This directory contains all image resources for the CTRC BattleBots curriculum.

## Directory Structure

```
images/
├── archetypes/          # Robot weapon archetype photos
│   ├── drum-spinner-*.jpg
│   ├── vertical-disc-*.jpg
│   ├── horizontal-bar-*.jpg
│   ├── eggbeater-*.jpg
│   ├── undercutter-*.jpg
│   └── midcutter-*.jpg
├── motors/              # N20 motor reference materials
│   ├── N20-motor-datasheet-GA12.pdf
│   └── n20-d-shaft-*.jpg
├── ATTRIBUTION.md       # Full attribution and licensing info
├── IMAGE_SOURCES.md     # Complete sourcing guide with download instructions
└── README.md           # This file
```

## Image Categories

### Robot Archetypes (5 types needed)

1. **Drum Spinner** - Rotating drum weapon
2. **Vertical Disc Spinner** - Vertical spinning disc weapon
3. **Horizontal Bar / Eggbeater** - Horizontal bar or multi-bar weapons
4. **Undercutter** - Weapon at ground level
5. **Midcutter** - Weapon between ground and top of chassis

### Motor References

1. **N20 Motor Datasheet** - Complete dimensional drawings
2. **D-Shaft Visual Reference** - Photos showing D-shaft fit

## Quick Start

### To Download Images

1. Open `IMAGE_SOURCES.md` for complete sourcing information
2. Visit each URL listed
3. Download images following the naming conventions
4. Save to appropriate subdirectory

### Naming Convention

```
[category]-[source]-[number].jpg

Examples:
- drum-spinner-alexkorvin-1.jpg
- vertical-disc-drizzle-1.jpg
- undercutter-lordtoby-1.jpg
- n20-d-shaft-fit-1.jpg
```

### Before Using Any Image

1. Check `ATTRIBUTION.md` for license and attribution requirements
2. Verify proper credit is given in curriculum materials
3. Ensure usage complies with educational fair use
