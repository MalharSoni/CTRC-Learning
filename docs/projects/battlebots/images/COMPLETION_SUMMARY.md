## Technical Details

### Diagram Format: ASCII Art in Markdown

**Why ASCII instead of image files?**
- ✅ Searchable and indexable
- ✅ Accessible to screen readers
- ✅ Easy to edit and maintain
- ✅ No file size concerns
- ✅ Version control friendly (git diff works)
- ✅ Responsive on all devices

**Example:**
```markdown
??? info "Power-On Sequence Flowchart"

    ```
    ┌────────────────────────────────┐
    │ 1. WEAPON LOCK INSTALLED       │
    │    ✓ Pin/zip-tie in place      │
    └──────────────┬─────────────────┘
                   │
                   ▼
    ┌────────────────────────────────┐
    │ 2. TRANSMITTER ON              │
    │    ✓ Turn on controller FIRST  │
    └──────────────┬─────────────────┘
    ```
```

**Renders as:** Clean, collapsible flowchart with visual flow

### Integration Method: Collapsible Info Blocks

**Uses MkDocs admonitions:**
```markdown
??? info "Diagram Title"
    Diagram content here (collapsible)

!!! note "Manual Task"
    Download instructions (always visible)
```

**Benefits:**
- Keeps pages clean (diagrams hidden until clicked)
- Important notes always visible
- Semantic HTML (proper accessibility)
