## Next Steps: Manual Download Required

### Why Manual Download?

Many sites (Thingiverse, Printables, Instructables) have bot protection preventing automated downloads. Images must be manually downloaded.

### Download Workflow

1. **Open IMAGE_SOURCES.md** - Contains all URLs and instructions
2. **Visit each URL** - Click through to source pages
3. **Download images** - Right-click and save
4. **Use naming convention:** `[archetype]-[source]-[number].jpg`
5. **Save to correct folder:**
   - Robot photos → `/archetypes/`
   - Motor materials → `/motors/`
6. **Update ATTRIBUTION.md** - Fill in "Date Accessed" fields
