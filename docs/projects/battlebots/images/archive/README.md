## Why Archived?

**Too Many Files:**
- Original approach created 9 separate documentation files
- Caused clutter and redundancy
- Hard to find information (scattered across files)

**Consolidated Approach:**
- 3 active files replace 9 archived files
- Clear separation: attribution.md (data) + VISUAL_ASSETS.md (guide) + LICENSE_COMPLIANCE.md (legal)
- Easier to maintain and update
