# Unit 1: Drivetrain Mastery

**The drivetrain is the foundation of every robot.** If your drivetrain is weak, wobbly, or slow, nothing else you build on top of it will work well. Unit 1 teaches you to build a drivetrain that is structurally sound, low-friction, and properly geared — from the ground up.

---

| | Details |
|---|---|
| **Modules** | 4 |
| **Field Challenges** | 2 |

## What You'll Learn

By the end of Unit 1, you will be able to:

- [x] Identify and correctly use every common VEX V5 part
- [x] Build structurally rigid frames using bracing and boxing techniques
- [x] Calculate and apply gear ratios for speed or torque
- [x] Minimize friction in rotating assemblies
- [x] Assemble a complete competition-quality drivetrain
- [x] Mount V5 electronics (Brain, Battery, Radio, Inertial Sensor)

## Tips for Success

!!! tip "The Golden Rule of Building"
    **If it wobbles, it's wrong.** A good drivetrain should feel solid in your hands. No rattling, no flex, no play where there shouldn't be play. If you pick it up and shake it, you should hear nothing.

!!! tip "Measure Twice, Build Once"
    Count your holes before you cut, drill, or attach. The hole pattern on VEX metal is your measurement system — every hole is exactly 0.5 inches apart. Use this to your advantage.
