# Unit 3: Onshape CAD

**Design it before you build it.** Onshape CAD is the tool competitive robotics teams use to design, iterate, and communicate their robot designs before a single piece of metal is cut. Unit 3 takes you from zero CAD experience to designing full VEX robot assemblies — all in the browser.

---

## Unit Overview

| | Details |
|---|---|
| **Modules** | 5 |
| **Field Challenges** | 3 |
| **Estimated Time** | 10–14 hours |
| **Final Deliverable** | A fully modeled, rendered drivetrain assembly with mates and gear relations |

---

## What You'll Learn

By the end of Unit 3, you will be able to:

- [x] Navigate the Onshape interface confidently
- [x] Create fully constrained 2D sketches
- [x] Use extrude, fillet, chamfer, and wrap features to create 3D parts
- [x] Design parts specifically for 3D printing
- [x] Import and use the VEX V5 Parts Library
- [x] Assemble parts using Fastened, Revolute, and Tangent mates
- [x] Use gear relations and replicate to build efficient assemblies
- [x] Render your designs for documentation and presentation

## Tips for Success

!!! tip "CAD Is a Skill, Not a Talent"
    Nobody is born knowing how to CAD. The students who get good at it are the ones who practice deliberately — not just following tutorials, but experimenting on their own. After each lesson, spend 10 minutes building something that wasn't assigned.

!!! tip "Constraints Are Your Best Friend"
    A fully constrained sketch turns black in Onshape. If your sketch lines are still blue, you have unconstrained geometry. Always fully constrain your sketches before extruding — it prevents unexpected behavior later.
