# Challenge: CAD Battle

**Speed. Accuracy. Clean modeling.** The CAD Battle tests your ability to observe a physical object, measure it, and reproduce it in Onshape as fast and accurately as possible. No tutorials. No hand-holding. Just you, a mystery object, and a 15-minute clock.

---

## Objective

Given a mystery physical object revealed by the coach, be the first to accurately model it in Onshape. Fastest accurate model wins.

---

## Format

| | Details |
|---|---|
| **Type** | Individual, timed |
| **Duration** | 15 minutes |
| **Rounds** | 1-3 (coach's discretion, escalating difficulty) |
| **Tools Allowed** | Onshape, calipers, ruler |
| **Tools NOT Allowed** | Importing existing models, copying from classmates |

---

## Rules

### Setup
1. All students open a **new, blank Onshape document** before the round starts
2. Name it `CAD-Battle-[YourName]-Round[X]`
3. Calipers and rulers are on the table
4. **Do not touch the mystery object until the coach says go**

### The Reveal
1. Coach places the mystery object in the center of the table
2. Coach says **"Go"** and starts the 15-minute timer
3. Students may approach the object, measure it, photograph it, and handle it
4. **Only one student may measure the object at a time** — form a line if needed, be respectful

### Mystery Object Types

The coach will select from objects of increasing difficulty:

| Difficulty | Example Objects |
|------------|----------------|
| **Easy** | A VEX standoff, a simple L-bracket, a spacer stack |
| **Medium** | A VEX gear, a motor cartridge, a custom 3D printed part |
| **Hard** | A household item (mug, stapler), a multi-feature VEX assembly |

!!! example "📷 Photo Needed"
    Example mystery objects at each difficulty: standoff, gear, household item

### During the Battle

- You may measure the object as many times as needed
- You may take photos of the object with your phone for reference
- You may **not** communicate with other students about dimensions or strategy
- You may **not** search the internet for existing models of the object
- Work in a single Part Studio — no imports

### Submission
1. When you are satisfied with your model, raise your hand
2. Coach notes your finish time
3. You may continue to refine until the 15-minute mark
4. At 15 minutes, all work stops — hands off keyboards

---

## Judging Criteria

Each submission is scored out of **100 points**:

| Criteria | Points | What Judges Look For |
|----------|--------|---------------------|
| **Dimensional Accuracy** | 40 | Key dimensions within 1mm of the real object |
| **Feature Accuracy** | 25 | Correct features used (holes, fillets, chamfers match the real object) |
| **Clean Modeling** | 20 | No extra sketches, no failed features, organized feature tree |
| **Speed Bonus** | 15 | Points awarded inversely by finish time (first = 15pts, second = 12pts, etc.) |

### Accuracy Measurement

The coach will compare your model to the real object by checking:

- Overall length, width, and height
- Hole diameters and positions
- Fillet/chamfer radii
- Feature count (did you miss any holes, cuts, or details?)

!!! info "Accuracy Tolerance"
    Dimensions within **1mm** of the actual measurement receive full credit. Dimensions within **2mm** receive half credit. Anything beyond 2mm receives no credit for that dimension.

## Strategy Guide

### The Winning Approach

!!! tip "Measure First, Sketch Second, Extrude Last"
    The biggest mistake in CAD Battle is rushing to Onshape before understanding the object. Spend the first 2-3 minutes measuring and noting key dimensions on paper. A student who measures for 3 minutes and models for 12 minutes will almost always beat a student who starts modeling immediately and has to remeasure repeatedly.

### Measurement Priority

1. **Bounding box** — overall length, width, height
2. **Primary features** — large holes, main cutouts, dominant geometry
3. **Secondary features** — small holes, fillets, chamfers, details
4. **Fine details** — only if you have time remaining

### Modeling Priority

1. **Base sketch** — the main cross-section shape, fully constrained
2. **Primary extrude** — get the 3D volume correct
3. **Major cuts** — large holes, main cutouts
4. **Fillets and chamfers** — add these last, they are easy to apply quickly
5. **Fine details** — only if ahead of the clock

### Time Management

| Phase | Time | Focus |
|-------|------|-------|
| Measure | 0:00 - 2:30 | All critical dimensions, written on paper |
| Base model | 2:30 - 7:00 | Main shape and primary features |
| Details | 7:00 - 12:00 | Holes, cuts, secondary features |
| Polish | 12:00 - 15:00 | Fillets, chamfers, clean up feature tree |

## Common Mistakes

| Mistake | Consequence | Prevention |
|---------|------------|-----------|
| Starting to model before measuring | Constant rework, wasted time | Write dimensions on paper first |
| Modeling features in the wrong order | Later features break earlier ones | Big shapes first, details last |
| Forgetting to constrain sketches | Dimensions shift when adding features | Constrain and dimension every sketch fully |
| Over-detailing early | Run out of time before capturing major features | Get the big picture right first |
| Guessing dimensions instead of measuring | Inaccurate model, low score | Always measure — never eyeball |
