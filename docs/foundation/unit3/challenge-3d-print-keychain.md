# Challenge: 3D Print Keychain

**Design something real, print something yours.** This challenge puts your Onshape skills to the test by asking you to design a custom keychain from scratch. The best part? You get to 3D print it and take it home.

---

## Objective

Design and 3D print a personalized keychain that showcases your CAD skills. This is a creative design challenge — there is no single correct answer.

---

## Requirements

Your keychain must include **all** of the following:

| Requirement | Details |
|-------------|---------|
| **Your name or initials** | Use the Text tool — must be legible on the printed part |
| **At least one fillet or chamfer** | Smooth or bevel at least two edges |
| **A creative shape** | Not a plain rectangle — show personality in the outline |
| **Keychain hole** | At least 4mm diameter, positioned for a keyring |
| **Printable dimensions** | No larger than 60mm x 40mm x 6mm |

### Feature Checklist

Use **at least 4** of these features in your design:

- [ ] Fully constrained sketch with dimensions
- [ ] Extrude-Add (base shape)
- [ ] Extrude-Remove (cutouts, holes, engraving)
- [ ] Text tool
- [ ] Fillet (rounded edges)
- [ ] Chamfer (beveled edges)
- [ ] Wrap (optional — for curved sections)
- [ ] Mirror or pattern (optional — for symmetry)

---

## Design Guidelines

### Shape Ideas

Do not just make a rectangle with your name on it. Consider:

- **Silhouette shapes** — a gear, a robot, a shield, your state outline
- **Rounded forms** — use arcs and fillets for an organic feel
- **Geometric patterns** — hexagons, interlocking shapes, tessellations
- **Functional shapes** — a bottle opener keychain, a phone stand, a mini tool

### Text Tips

- Minimum text height: **8mm** for readability after printing
- Raised text (Extrude-Add) is easier to read than engraved text on small parts
- If using engraved text, set depth to at least **0.8mm**
- Sans-serif fonts print cleaner than serif fonts at small sizes

!!! example "🖥️ Screenshot Needed"
    Raised text (Extrude-Add) vs engraved text (Extrude-Remove) on keychain

### Printability Reminders

- Minimum wall thickness: **1.2mm**
- No unsupported overhangs greater than 45 degrees
- The keychain hole needs a **bridge** across the top if it is near the edge — make sure there is at least 2mm of material around it
- Flat bottom surface = best print quality

## Process

### Step 1: Sketch It on Paper First (5 minutes)

Before opening Onshape, sketch your idea on paper. This is faster than iterating in CAD and helps you make creative decisions without getting bogged down in tool mechanics.

### Step 2: Model It in Onshape (30-45 minutes)

1. Create a new document named `[YourName]-Keychain`
2. Build your design using the techniques from Modules 3.1 and 3.2
3. Fully constrain all sketches
4. Apply fillets/chamfers last (they are easier to modify if applied at the end)

### Step 3: Coach Review (5 minutes)

Before printing, have your coach review for:

- Printability (wall thickness, overhangs, dimensions)
- Feature usage (did you meet the requirements?)
- Structural integrity (will it survive being on a keyring?)

### Step 4: Export and Print

1. Right-click the Part in the feature tree
2. Select **Export** > **STL**
3. Use default settings (Fine resolution)
4. Hand the STL file to your coach for slicing and printing

### Step 5: Post-Print

- Remove support material if any
- Light sanding if needed
- Thread a keyring through the hole
- It is yours to keep

## Judging

!!! note "This Is Not a Competition — It Is a Celebration"
    Everyone gets to print their keychain. There are no losers. The class vote is just for fun and to celebrate creative design thinking.

### Class Vote Categories

| Category | What It Means |
|----------|--------------|
| **Most Creative Shape** | The most interesting or unexpected outline |
| **Best Use of Features** | The design that uses the most CAD features effectively |
| **Cleanest Print** | The design that printed the best (good design-for-manufacturing) |
| **Most Likely to Actually Use** | The keychain you would actually put on your keys |

### How Voting Works

1. All printed keychains are displayed on a table
2. Each student gets **2 votes** (you cannot vote for yourself)
3. Votes are anonymous
4. Winners are announced and recognized — bragging rights only

## Common Mistakes to Avoid

| Mistake | Why It Is a Problem | Fix |
|---------|-------------------|-----|
| Text too small | Unreadable after printing | Use 8mm+ height |
| Walls too thin | Breaks during or after printing | 1.2mm minimum |
| No keychain hole | Cannot attach to a keyring | Add 4mm+ hole |
| Unconstrained sketch | Part dimensions change unexpectedly | Fully constrain before extruding |
| Too complex for first design | Frustration, long print time | Start simple, add detail if time allows |
