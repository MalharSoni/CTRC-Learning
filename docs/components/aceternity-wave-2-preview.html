Aceternity UI - Wave 2 Components

Three new components adapted for CTRC Learning: Animated Modal, Sticky Scroll Reveal, and Hover Border Gradient

1. Animated Modal
A customizable modal with smooth animations, outside-click detection, and compound component pattern.
Framer Motion Compound Pattern Phosphor Icons 
import { Modal, ModalTrigger, ModalBody, ModalContent, ModalFooter } from "./animated-modal";

<Modal>
  <ModalTrigger>Open Modal</ModalTrigger>
  <ModalBody>
    <ModalContent>
      <h2>Your Content Here</h2>
      <p>Modal body content...</p>
    </ModalContent>
    <ModalFooter>
      <button>Action</button>
    </ModalFooter>
  </ModalBody>
</Modal>

Smooth Animations

Spring-based animations for natural feel using Framer Motion

Outside Click

Automatically closes when clicking outside the modal

Flexible Content

Use ModalContent and ModalFooter for structured layouts

Dark Theme

Styled for CTRC's dark theme with accent color

import { HoverBorderGradient } from "./hover-border-gradient";

<HoverBorderGradient
  containerClassName="rounded-full"
  duration={1}
  clockwise={true}
  className="bg-[var(--bg-dark-2)] text-white"
>
  Button Text
</HoverBorderGradient>

Rotating Gradient

Animated border gradient rotates around the button

Configurable Speed

Control animation duration (default 1s)

Direction Control

Rotate clockwise or counter-clockwise

Flexible Styling

Works with any border radius (rounded-full, rounded-md, etc.)
