1️⃣ Animated Modal

Compound Components Framer Motion

Basic Usage

import {
  Modal,
  ModalTrigger,
  ModalBody,
  ModalContent,
  ModalFooter,
} from "./components/animated-modal";

export function MyModal() {
  return (
    <Modal>
      <ModalTrigger>
        Open Modal
      </ModalTrigger>

      <ModalBody>
        <ModalContent>
          <h2 className="text-2xl font-bold text-white mb-4">
            Modal Title
          </h2>
          <p className="text-neutral-400">
            Your content goes here...
          </p>
        </ModalContent>

        <ModalFooter>
          <button>Cancel</button>
          <button>Confirm</button>
        </ModalFooter>
      </ModalBody>
    </Modal>
  );
}

With Animated Trigger

import { RocketLaunch } from "@phosphor-icons/react";

<ModalTrigger className="flex justify-center group/modal-btn">
  <span className="group-hover/modal-btn:translate-x-40 transition duration-500">
    Click Me
  </span>
  <div className="-translate-x-40 group-hover/modal-btn:translate-x-0 absolute inset-0 flex items-center justify-center transition duration-500">
    <RocketLaunch size={20} weight="bold" />
  </div>
</ModalTrigger>

Props

Component Props Description
ModalTrigger children, className Button to open modal
ModalBody children, className Modal container
ModalContent children, className Main content area
ModalFooter children, className Footer with actions
✓ Auto Features: Outside-click detection, Escape key, body scroll lock, backdrop blur.

2️⃣ Sticky Scroll Reveal

Scroll-Based Sticky Position

Basic Usage

import { StickyScrollReveal } from "./components/sticky-scroll-reveal";

const content = [
  {
    title: "Section 1",
    description: "First section description...",
    content: (
      <div className="flex items-center justify-center h-full">
        <p>Visual content here</p>
      </div>
    ),
  },
  {
    title: "Section 2",
    description: "Second section description...",
    content: (
      <div>Different visual</div>
    ),
  },
  // Add 2-4 more sections...
];

export function MyScroll() {
  return <StickyScrollReveal content={content} />;
}

With Icons

import { Code, Robot, Trophy } from "@phosphor-icons/react";

const content = [
  {
    title: "Learn to Code",
    description: "Master C++ programming...",
    content: (
      <div className="h-full flex items-center justify-center">
        <Code size={80} weight="duotone" className="text-[var(--accent)]" />
      </div>
    ),
  },
  // ... more sections
];

Props

Prop Type Description
content StickyScrollContent[] Array of section objects (required)
contentClassName string Classes for sticky visual container (optional)
⚠ Best Practice: Use 3-5 sections max. Too many sections make scrolling tedious.

3️⃣ Hover Border Gradient

Gradient Animation Hover Effect

Basic Usage

import { HoverBorderGradient } from "./components/hover-border-gradient";

export function MyButton() {
  return (
    <HoverBorderGradient
      containerClassName="rounded-full"
      className="bg-[var(--bg-dark-2)] text-white"
    >
      Click Me
    </HoverBorderGradient>
  );
}

With Icons & Custom Duration

import { ArrowRight } from "@phosphor-icons/react";

<HoverBorderGradient
  containerClassName="rounded-full"
  duration={2}              // 2 second rotation
  clockwise={false}         // Counter-clockwise
  className="bg-[var(--bg-dark-2)] text-white flex items-center gap-2"
>
  <span>Get Started</span>
  <ArrowRight size={16} weight="bold" />
</HoverBorderGradient>

As a Link (Not Button)

<HoverBorderGradient
  as="a"
  href="/courses"
  containerClassName="rounded-md"
  className="bg-[var(--bg-dark-2)] text-white"
>
  View Courses
</HoverBorderGradient>

Props

Prop Type Default Description
children ReactNode - Button content
containerClassName string - Outer container classes
className string - Inner content classes
as ElementType "button" HTML element type
duration number 1 Animation duration (seconds)
clockwise boolean true Rotation direction
✓ Versatile: Works with rounded-full, rounded-md, rounded-lg, or any border radius.

🎨 Common Patterns

Pattern 1: Modal with Feature Grid

<ModalContent>
  <h2>Features</h2>
  <div className="grid grid-cols-2 gap-4">
    {features.map((feature, idx) => (
      <motion.div
        key={idx}
        initial={{ opacity: 0, y: 20 }}
        animate={{ opacity: 1, y: 0 }}
        transition={{ delay: idx * 0.1 }}
        className="p-4 bg-[var(--bg-dark-3)] rounded-lg"
      >
        {feature.icon}
        <h3>{feature.title}</h3>
        <p>{feature.description}</p>
      </motion.div>
    ))}
  </div>
</ModalContent>

Pattern 2: Sticky Scroll with Images

content: (
  <div className="h-full w-full flex items-center justify-center p-8">
    <img
      src="/path/to/image.jpg"
      alt="Description"
      className="rounded-lg object-cover w-full h-full"
    />
  </div>
)

Pattern 3: CTA Button Group

<div className="flex gap-4">
  <button className="px-4 py-2 rounded-md border border-[var(--border)] text-neutral-400">
    Secondary Action
  </button>

  <HoverBorderGradient containerClassName="rounded-md">
    Primary Action
  </HoverBorderGradient>
</div>

💡 Pro Tips

  1. Modal: Keep content under 600px height to avoid scroll issues
  2. Sticky Scroll: Use 3-5 sections for optimal UX (not too long)
  3. Hover Gradient: Use sparingly - not every button needs it!
  4. Performance: Lazy load heavy modal content
  5. Accessibility: Always provide meaningful alt text for images
  6. Mobile: Test all components on mobile - sticky scroll hides visual on small screens
