Floating Dock Component

macOS-style navigation dock with smooth hover animations. Hover over icons to see the magnification effect.

Home Learn Projects Design Build Compete Community

Usage Example

Import and use the FloatingDock component in your React application:

import { FloatingDock } from './floating-dock'
import { House, BookOpen, Trophy } from '@phosphor-icons/react'

const links = [
  { title: "Home", icon: <House size={20} />, href: "/" },
  { title: "Learn", icon: <BookOpen size={20} />, href: "/lessons" },
  { title: "Compete", icon: <Trophy size={20} />, href: "/competition" },
]

<FloatingDock items={links} />

Features

• Smooth magnification effect on hover
• Responsive desktop and mobile variants
• Phosphor Icons integration
• CTRC design system colors
• Backdrop blur for glass effect
• Tooltip labels on hover

Installation

npm install @phosphor-icons/react framer-motion
